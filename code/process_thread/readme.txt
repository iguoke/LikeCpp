多进程和多线程的

一 多进程
	概念：是计算机的程序关于某数据集合上的一次运行，是系统资源分配和调度的基本单位
	是一个实体，包括
		文本区：处理器执行的代码
		数据区：变量和动态内存
		堆栈:调用的指令和本地 变量
	特征：
		动态
		并发
		独立
		异步
		结构
	fork（）函数
		Unix系统最杰出的成就之一
		执行一次返回两个值
	exec类函数
		程序本身“死掉”，新程序运行，对系统而言还是同一个进程
	fork和exec结合用

二Linux下进程间通信（IPC）
	管道
		无名管道：父子进程
			pipe（）
		有名管道：同一机器任意进程
			mkfifo（“myfifo”，“rw”；
			mknod myfifo p；
	内存共享
		同一机器间通信
		shmXXX函数
	信号量
	socket

		
三多线程
	机制
		进程模拟的
		是进程克隆了父进程的资源
	优点：
		简化处理异步事件
		共享数据
		多任务
	pthread创建
	线程同步
		互斥
		读写锁
		条件变量
